# Music classifier

Ce projet propose un **classificateur de musique** implémenté en Python utilisant la regression logistique et la descente de gradient stochastique.
Il contient un script principal (`projet.py`) qui effectue l’entraînement et/ou la classification de données musicales et un rapport (`report.pdf`) avec l’analyse des résultats.

## Contenu

- `projet.py` — script Python principal pour entraîner et tester le modèle.
- `report.pdf` — rapport décrivant la méthodologie et les résultats.
